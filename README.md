# scheme-evaluator

A toy scheme evaluator implemented by python3, features:

- varible and function
- expression
- lambda expression
- closure

## How to run

Run `evaluator.py`ï¼š

```scheme
--------------------

(define (add-six x) (+ x 6))
ok

End Evaluation
--------------------

--------------------

(add-six x)
40

End Evaluation
--------------------
```
